# New Bedford City Codes Search

This application provides a searchable interface for New Bedford city codes, with AI-powered explanations and the ability to ask questions.

## Contact Information

For any questions or support regarding this application, please contact:

- Developer: Joseph Barney
- Email: Joseph1914@gmail.com

## Features

- Search New Bedford city codes
- AI-powered explanations of code sections
- Ability to follow up on the explanation and ask the ai about the code.
- User-friendly interface

## Technical Stack

- Frontend: React.js
- Backend: Flask (Python)
- Database: MongoDB
- Deployment: Netlify (Frontend), Heroku (Backend)
